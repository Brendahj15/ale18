Semana 01

A continuación, te comentaré sobre mis primeros aprendizajes en Laboratorio 2.

Clases presenciales

Siendo las 7 p.m un jueves, empezamos a interactuar entre todos. El profesor anunció que debíamos presentarnos uno por uno para poder conocer nuestros nombres, habilidades, hobbies y alguna u otra anécdota que tengamos acerca de nosotros. Aprendimos que todos somos diferentes y podemos complementarnos con las distintos talentos que tenemos y que no es malo ser vulnerables.

Clases virtuales

En esta clase, conocí por primera vez la web Github y el programa Visual Studio para configurar mi primera página web con código HTML descargando plantillas de Netlify. Primero cree mi cuenta en Github, aprendí acerca de Commit Changes que son los cambios que realizamos entre varias personas en el Visual Studio para la edición de la página web. Sincronizamos con GH-PAGES, al comienzo no me aparecía mi link de página pero al revisar la clase grabada pude lograrlo.
