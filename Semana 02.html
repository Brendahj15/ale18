Clases virtuales

Aquí seguimos explorando las diferentes plantillas de Netlify para nuestro desarrollo de bitácora virtual, empezando a aprender los diferentes códigos para su edición. Aprendí lo que es index.html que es el archivo base, cree mi carpeta en disco C para mi página web de Github, colocar siempre AUTOSAVE para que los cambios no se pierdan. Aprendimos a trabajar sobre la rama GH-PAGES.
